# Closure Helpers

Generic helpers for the [Closure Library][closure-library].

## Getting Started

If you have [bower][] installed:
```shell
bower install closure-structs --save-dev
```

## Documentation

Sorry, no docs available, for now read the well documented included files:

* [helpers.js](lib/helpers.js) Generic function helpers.
* [debug.js](lib/debug.js) Thin debug wrapper.
